# REST API specification for URL shortener project

This is an [OpenAPI](https://www.openapis.org/) specification for a URL shortener project.

## Tools

### Option #1
IntelliJ IDE ([Idea EDU](https://www.jetbrains.com/edu-products/download/#section=idea) will work) + 42Crunch [OpenAPI (Swagger) Editor](https://plugins.jetbrains.com/plugin/14837-openapi-swagger-editor) plugin

### Option #2
[VS Code](https://code.visualstudio.com/) + [OpenAPI (Swagger) Editor](https://marketplace.visualstudio.com/items?itemName=42Crunch.vscode-openapi) extension

## Example

This repo already has an OpenAPI template with a simple example but if you need a more sophisticated one you could find it [here](https://github.com/OAI/OpenAPI-Specification/blob/master/examples/v3.0/petstore.yaml).
